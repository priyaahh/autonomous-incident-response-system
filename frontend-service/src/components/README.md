# Components Directory

Place reusable UI controls, presentation cards, and custom canvas nodes here.

## Proposed Components
1. **AgentGraphCanvas.jsx**: Renders the dynamic workflow graph representing LangGraph nodes and active state edges (using custom SVGs or React Flow).
2. **AlertPanel.jsx**: Displays real-time inbound system alarms with severity filtering.
3. **IncidentReportViewer.jsx**: Markdown renderer displaying generated post-mortems with code highlight formatting.
4. **LogConsole.jsx**: Visual console output with trace levels and search parameters.
