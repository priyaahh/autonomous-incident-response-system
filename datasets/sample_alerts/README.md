# Sample Alerts Directory

This folder is used to collect standard alert schemas generated by production tools (Prometheus Alertmanager, AWS CloudWatch, Datadog) to test the backend API receivers.

---

## 🗂️ Sample Prometheus Alertmanager Webhook Payload: `prometheus_alert.json`

```json
{
  "receiver": "airs-webhook-handler",
  "status": "firing",
  "alerts": [
    {
      "status": "firing",
      "labels": {
        "alertname": "PostgresConnectionPoolExhausted",
        "severity": "critical",
        "service": "postgres-db",
        "instance": "db-node-02.internal.net"
      },
      "annotations": {
        "summary": "Database connection pool utilization has exceeded 95% on instance db-node-02",
        "description": "Active connections count is 98/100 for over 5 minutes. High latency observed on dependent APIs.",
        "dashboard_link": "https://grafana.internal.net/d/postgres-metrics"
      },
      "startsAt": "2026-06-17T12:00:00.000Z",
      "generatorURL": "https://prometheus.internal.net/graph?g0.expr=pool_utilization+%3E+95"
    }
  ],
  "groupLabels": {
    "alertname": "PostgresConnectionPoolExhausted"
  },
  "commonLabels": {
    "severity": "critical"
  },
  "commonAnnotations": {
    "summary": "Database connection pool utilization has exceeded 95%"
  }
}
```
